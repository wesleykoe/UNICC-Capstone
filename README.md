# UNICC AI Safety SLM Platform

Local Small Language Model (SLM) platform prototype for the UNICC Capstone project.

## Features

- Local SLM text generation
- FastAPI server
- /health endpoint
- /generate endpoint
- Reproducible environment

## Setup

```bash
git clone https://github.com/wesleykoe/UNICC-Capstone
cd unicc-slm
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
